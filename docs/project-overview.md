# AI Document Controller Assistant - Project Overview

## Introduction

AI Document Controller Assistant is an industry-based n8n automation workflow designed for construction, engineering consultancy, and project document control teams.

The workflow automates the complete document control process from Gmail document receiving to Google Drive storage, Google Sheets logging, approval tracking, and Gmail draft replies.

## Problem Statement

In construction and engineering companies, document controllers receive many project documents through email.

These documents may include:

- Material Submittals
- RFIs
- MIRs
- WIRs
- NCRs
- Drawings
- Method Statements
- Transmittals

Manual document control tasks can be time-consuming and error-prone.

## Main Objective

The main objective of this project is to automate document control operations using n8n, AI, Gmail, Google Drive, and Google Sheets.

## Key Features

- Gmail email detection
- Attachment checking
- PDF text extraction
- AI metadata extraction
- Missing information detection
- Duplicate checking
- Revision tracking
- Google Drive folder routing
- Google Sheets document register
- Approval status tracking
- Gmail draft reply creation
- Email mark as read

## Tools and Technologies

- n8n
- Gmail
- Google Drive
- Google Sheets
- AI Information Extractor
- Groq Chat Model
- JavaScript Code Nodes

## Real-World Use Case

This workflow is useful for:

The workflow successfully automates the document control lifecycle and reduces repetitive manual work.
